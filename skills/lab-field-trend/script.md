# lab-field-trend — Execution Script

## Step 1: Load config and context

```bash
cat ~/.openclaw/workspace/lab/LAB_CONFIG.json
```

Extract:
- `fields` and `databases` → search scope
- `obsidian_vault` → save path
- `notify_channel` → delivery target
- `weekly_trend_day/time` → for cron confirmation

Also read `research-graph.jsonl` to extract:
- All active Project nodes → their names + keywords
- All existing Paper node IDs → to deduplicate results

---

## Step 2: Build search queries

For each field in `fields`, generate search queries:
- Primary: field name + "last 7 days"
- Secondary: extract keyword clusters from active project hypotheses in research graph

Example queries for fields: ["neuroscience", "speech processing"]:
```
neuroscience[MeSH] AND ("last 7 days"[PDat])
speech perception[MeSH] AND ("last 7 days"[PDat])
neural coupling AND speech AND 2025[DP]
```

---

## Step 3: Query APIs

### PubMed (via NCBI E-utilities — no API key required, key optional for higher rate limits)

```bash
# Search
curl "https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi?db=pubmed&term=<query>&datetype=pdat&reldate=7&retmax=50&retmode=json"

# Fetch abstracts
curl "https://eutils.ncbi.nlm.nih.gov/entrez/eutils/efetch.fcgi?db=pubmed&id=<ids>&rettype=abstract&retmode=xml"
```

### OpenAlex (free, no key required)

```bash
curl "https://api.openalex.org/works?filter=concepts.display_name:<field>,from_publication_date:<7-days-ago>&sort=cited_by_count:desc&per-page=50"
```

### arXiv (for CS/ML/physics fields)

```bash
curl "https://export.arxiv.org/api/query?search_query=all:<field>&start=0&max_results=30&sortBy=submittedDate&sortOrder=descending"
```

Deduplicate results against existing Paper nodes in research-graph.jsonl by DOI or title similarity.

---

## Step 4: Score and rank papers

Score each paper (0–100):

| Signal | Weight | How |
|---|---|---|
| Field keyword match | 30 | count matches in title/abstract |
| Project hypothesis match | 25 | semantic overlap with hypothesis text |
| Citation velocity | 20 | cited_by_count / days_since_publish (OpenAlex) |
| Method novelty | 15 | presence of "novel", "first", "new approach" in abstract |
| Journal impact | 10 | known high-impact journal list |

Keep top 20 by score.

---

## Step 5: LLM clustering and analysis

Send top 20 paper titles + abstracts to LLM with this prompt:

```
You are a research assistant. Given these {n} papers published this week in {fields}:

1. Group them into 3-5 thematic clusters. Name each cluster.
2. Identify the top 3 most significant breakthroughs (explain why in 1 sentence each).
3. Identify 1-2 emerging methods appearing across multiple papers.
4. Flag any papers that directly contradict or challenge these existing hypotheses:
   {list of active project hypotheses from research graph}
5. Identify 1 underexplored gap: something the field SHOULD be studying but isn't.

Be concise. No filler. Each point max 2 sentences.

Papers:
{title, authors, abstract for each}
```

## Error handling

- **API timeout/error:** Retry once. If still fails, skip that source, note in digest, continue with others.
- **No results:** Broaden query to 14 days, note in digest.
- **LLM clustering fails:** Skip clustering, just list top papers by score.
- **Obsidian vault not found:** Save digest to `~/.openclaw/workspace/lab/digests/{date}.md` and warn user.
